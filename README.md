## Getting Started

To get started with this portfolio template, follow these simple steps:

### Prerequisites

Make sure you have node `16.0.0` or higher and npm `9.0.0` or higher installed.

### Setup

- With git, clone the code to your machine, or download a ZIP of all the files directly.
- [Download the ZIP file from this location](https://github.com/prathmeshpote99/Portfolio.git)
```
git clone https://github.com/prathmeshpote99/Portfolio.git
```

- Once the files are on your machine, open the **Portfolio-Template** folder in [Visual Studio Code](https://code.visualstudio.com/download).

```
cd Portfolio-Template && code .
```

## Install

- After opening the files in Visual Studio Code, open the **VS Code** integrated terminal and run the following commands:

```
npm install
```

This will install all the packages and dependencies used in the project.

## Usage

- Run the following command to start a local server:

```
npm run dev
```

This will open up the project on a browser on `http://localhost:5173/`

## Build

- To create a production build:

```
npm run build
```

- To preview the site as it will appear once deployed:

```
npm run preview
```
